# Omniauth::Frederick

* An omniauth strategy for Frederick

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-frederick'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-frederick
